Windows Watch Errors

Common Windows system file watching errors observed:

  • DumpStack.log.tmp
  • hiberfil.sys
  • pagefile.sys
  • swapfile.sys

Solution: These can be safely ignored as they're Windows system files. Add to watchOptions exclude in next.config.js.
