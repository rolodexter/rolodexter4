Define and implement a modern, minimalist UI direction for the Rolodexter4 homepage, inspired by high-quality UI patterns and incorporating dynamic, interactive elements.

üéØ Core Design Elements

  • Character Stats-Style Interface:
    • Circular character/agent portrait frames with level indicators
    • Hexagonal/geometric stat indicators (like the JACK UI example)
    • Numerical stats with icon indicators (Strength, Defense, etc.)
    • Equipment/loadout slots with hover details
  • Dark Gaming Aesthetic:
    • Hunt: Showdown-inspired dark theme with red accents
    • Geometric frame decorations around UI elements
    • Diagonal line patterns and tech overlays
    • Atmospheric background images with fog/particle effects
  • Game UI Components:
    • Daily reward/task grid (like the Chinese game UI example)
    • Supply crate-style info boxes with diagonal markers
    • Character/agent selection grid with portraits
    • Minimal HUD-style top navigation
